Timestamp Microservice

Date timestamp:

Covert incoming date request to Unix and display in UTC format.
Type in the address bar preferred date using the syntax below.

Syntax:

[project url]/api/date     (date: YYYY/MM/DD)


Endpoint:

http://localhost:port /api/2022-11-01


Output:

{"unix":1667260800000,   "utc":"date in utc"}

Unix timestamp:

Covert incoming unix request to date and display in UTC format.
Type in the address bar a unix time using the syntax below.

Syntax:

[project url]/api/unix     (unix: 13-digits)


Endpoint:

http://localhost:port /api/1671926400000


Output:

{"unix": 13 digit time,   "utc":"Sun, 25 Dec 2022 00:00:00 GMT"}
